# Mensageria  
A mensageria se refere ao processo de envio e recebimento de uma mensagem entre diferentes componentes de um sistema ou em sistemas distintos. É um processo crucial para sistemas distribuídos onde certas aplicações precisam funcionar de maneira assíncrona, onde não é necessário ter alguém para receber e responder no exato momento. A mensageria facilita a troca de mensagens e garante que sejam entregues de maneira confiável, mesmo quando um dos sistemas estejam temporariamente indisponíveis. 

## Exemplo de mensageria 

### Sistemas de E-Commerce 

Em um sistema de E-Commerce ele fica responsável por processar milhares de pedidos, enviar mensagens de confirmação de compra, verificação de estoque e a atualização do status de pedido 

### Como a mensageria funciona no E-Commerce

- Quando o pedido é confirmado, o sistema de e-commerce pode enviar uma mensagem para uma fila como o Amazon SQS. 

- Um serviço de processamento de pedidos consome essa mensagem e inicia o processamento do pedido: verificando estoque, confirmando o pagamento, e atualizando o status do pedido. 

- Se o sistema de pagamento estiver temporariamente indisponível, a mensagem permanecerá na fila até que possa ser processada, garantindo que nenhum pedido seja perdido. 

### Monitoramento de Sensores em IoT 

Em um ambiente de IoT (Internet das Coisas), locais onde os sensores são instalados em diferentes pontos podem enviar dados continuamente, como temperatura, umidade, e presença de movimento. 

### Como a mensageria funciona em sensores IoT: 

- Cada sensor envia seus dados para um broker de mensagens como Apache Kafka. 

- Os dados são organizados em tópicos, por exemplo, um tópico para dados de temperatura e outro para umidade. 

- Diferentes serviços podem se inscrever nesses tópicos para processar e analisar os dados, como um serviço que alerta quando a temperatura ultrapassa um limite seguro. 

 
### Processamento de Transações Bancárias 

Os bancos lidam com milhões de transações diariamente, onde cada transação deve ser registrada, validada e confirmada. 

### Como a mensageria funciona em transações bancárias 

- Cada transação é enviada como uma mensagem para um broker de mensagens como o RabbitMQ. 

- Um serviço de processamento consome essas mensagens para validar as transações, atualizar saldos e registrar as operações. 

- A mensageria garante que, mesmo em caso de falha de um dos serviços, as transações sejam processadas corretamente quando o serviço estiver disponível novamente. 

## Benefícios da Mensageria 

- Desacoplamento: Assim os componentes do sistema não precisam estar diretamente conectados ou saber detalhes um do outro, o que facilita a escalabilidade e manutenção. 

- Escalabilidade: As mensagens podem ser enfileiradas e processadas por múltiplas instâncias de um serviço, permitindo que o sistema lide com grandes volumes de dados. 

- Confiabilidade: Garante que as mensagens sejam entregues e processadas mesmo em caso de falhas temporárias, evitando a perda de dados. 

## Tecnologias Comuns 

- Apache Kafka: Ideal para streaming de dados em grande escala e processamento em tempo real. 

- RabbitMQ: Utilizado para enfileiramento de mensagens com suporte a vários protocolos. 

- Amazon SQS: Serviço de fila de mensagens gerenciado que simplifica a comunicação entre diferentes partes de uma aplicação na AWS.

# Big Data

Um Big Data se refere a um conjunto de dados extremamente grandes e complexos, que não é possivel analisar ou processar esses dados através de um metodo tradicional de processamento de dados. Assim ele se torna um conceito fundamental em varias industrias modernas, onde a capacidade de analisar grandes volumes de dados em tempo real pode fornecer insights valiosos, influenciar decisões no negocio e melhorar a eficiencia operacional.

## Principais caracteristicas de Big Data

- **Volume:** É o que se refere a quantidade massiva de dados gerados a cada segundo por diversas fontes sendo em petabytes ou até exabytes, podendo tanto ser extraido de redes sociais, IoT, transações financeiras e muitas outras que possam gerar dados.

- **Variedade:** Pode vir de qualquer tipo de dado, sendo eles estruturados como por exemplo os bancos de dados, não estruturados sendo videos, imagens e texto e semi estruturados como os logs.

- **Velocidade:** Se refere a velocidade em que os dados podem ser gerados e processados, sendo por exemplo o fluxo de dados sensores em tempo real.

- **Veracidade:** Se trata da confiabilidade dos dados, sendo assim é necessário que tenha qualidade e precisão nas informações coletadas.

- **Valor:** Representa a capacidade de extrair valor e insights de dados, assim transformando esses dados em informações úteis para tomada de decisões.

## Exemplos de Big Data

### Redes Sociais
As plataformas como Facebook, Instagram e Twitter são aplicações que geram grande quantidade de dados, com milhões de post, likes, comentarios e intereções.

- **Uso do Big Data:** As empresas podem pegar esses dados gerados de suas aplicações e analisar o comportamento de seus usuários, tendências de mercado e segimentação de anuncios. Por exemplo, o Facebook utiliza big data para personalizar o feed com noticias e segmentar anuncios e publicações baseadas no interesse de seus usúarios.

### Setor da saúde

Hospitais e clinicas de saúde que geram grande quatidade de dados com, médicos eletrônicos, imagens de diagnóstico e os dispositivos de monitoramento dos pacientes.

- **Uso do Big Data:** Usando o big data é possivel detectar padrões em grandes populações de pacientes, prever surtos de doenças, personalizar os tratamento e até mesmo realizar diagnósticos precoces. Por exemplo, o uso do Big Data foi necessario para o rastreamento e resposta da pandemia de COVID-19, ajudando a modelar na propagação do vírus.

### Comércio eletrônico

Sites de E-Commerce, como Amazon e Alibaba, são sites que lidam com milhões de transações de usuários, incluindo buscas, compras e avaliações de produtos.

- **Uso do Big Data:** Esses dados são utilizados para recomendar produtos para seus clientes com base em suas preferências e comportamentos passados, melhorar a experiência do usuário e otimizar a cadeia de suprimento. Por exemplo, a Amazon utiliza o Big Data

### Manufatura

As industrias de manufatura geram dados através de maquinas, sensores e sistemas de controle de produção.

- **Uso no Big Data:** Ele pode contribuir na analise desses dados para manutenção preditiva, onde as falhas em maquinas podem ser previstas antes que ocorram, aumentando assim a eficiência operacional e reduzindo os custos de manutenção. Por exemplo, empresas como a General Eletric(GE) utilizam o Big Data para monitorar e otimizar a performance de turbinas, motores de aeronaves e outras maquinas industriais.

### Serviços financeiros

Bancos e instituições financeiras processam enormes volumes de transações e interações de clientes diarimente.

- **Uso do Big Data:** O uso da Big data ajuda na detecção de fraudes em tempo real, analise de créditos e personalização de produtos financeiros. Por exemplo, o uso de algoritimos de Big Data permite identificar padrões de fraudes em transações bancárias e bloquear transações suspeitas instantaneamente.

## O que é utilizado no Big Data

### Armazenamento de Dados

   - **Hadoop Distributed File System (HDFS)**: Um sistema de arquivos distribuído que armazena grandes volumes de dados em clusters de servidores.

   - **Amazon S3**: Um serviço de armazenamento de objetos da Amazon Web Services, amplamente usado para armazenar grandes volumes de dados não estruturados.

   - **Google Cloud Storage**: Similar ao Amazon S3, é utilizado para armazenar grandes conjuntos de dados na nuvem.

### Processamento de Dados

   - **Apache Hadoop**: Um framework que permite o processamento de grandes conjuntos de dados em um ambiente distribuído.

   - **Apache Spark**: Uma ferramenta de processamento de dados que executa tarefas em memória, oferecendo um desempenho muito mais rápido do que o Hadoop para certas operações.

   - **Apache Flink**: Projetado para processamento de fluxo de dados em tempo real e análise em tempo real.

### Análise de Dados

   - **Apache Hive**: Uma ferramenta de data warehousing construída sobre o Hadoop que permite consultar e analisar dados usando uma linguagem semelhante ao SQL.

   - **Apache HBase**: Um banco de dados NoSQL usado para armazenar dados não estruturados e processá-los em tempo real.

   - **Elasticsearch**: Um motor de busca distribuído que permite a indexação e pesquisa em tempo real de grandes volumes de dados.

### Ferramentas de Integração de Dados

   - **Apache Kafka**: Um sistema de mensagens distribuídas usado para construir pipelines de dados e processar fluxos de dados em tempo real.

   - **Apache NiFi**: Uma ferramenta para automação e integração de dados que permite o fluxo de dados entre sistemas diferentes de forma segura e escalável.

   - **Apache Airflow**: Um orquestrador de workflows que permite a criação, agendamento e monitoramento de pipelines de dados.

### Gerenciamento e Visualização de Dados

   - **Tableau**: Uma ferramenta de visualização de dados que permite criar gráficos interativos e painéis a partir de grandes volumes de dados.

   - **Power BI**: Uma ferramenta de business intelligence da Microsoft que ajuda a visualizar e compartilhar insights extraídos dos dados.

   - **Grafana**: Usado para criar dashboards interativos e monitorar a saúde dos sistemas em tempo real.

### Segurança e Governança de Dados

   - **Apache Ranger**: Um framework para gerenciar a segurança e as políticas de acesso a dados em um cluster Hadoop.

   - **Apache Atlas**: Uma ferramenta de governança de dados que oferece funcionalidades como catalogação, linhagem de dados e gerenciamento de metadados.
